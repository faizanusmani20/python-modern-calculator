# Modern Python Calculator 🖩

A simple command-line calculator built with Python using the modern `match-case` statement introduced in Python 3.10.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Exponentiation (`**`)
- Division-by-zero handling
- Invalid operator handling
- Interactive loop for repeated calculations

---

## Requirements

- Python 3.10 or higher
